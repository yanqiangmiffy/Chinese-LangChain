# chinese-langchain

> Chinese-LangChain：中文langchain，基于ChatGLM-6b+langchain实现本地化知识库检索与智能答案生成

## 特性

- 支持多种文档上传与内容解析：pdf、docx，ppt等
- 支持知识增量更新

[//]: # (- 支持检索结果与LLM生成结果对比)
